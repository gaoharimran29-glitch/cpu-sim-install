
# cpu-sim-install
Automated Script to download cpusim
Follow these steps to install cpusim , run this script ( made by me) guys. (Only for windows)
First download it and then right click on it and choose the option 'run as administrator'. 
Delete any java version you previously download. 
Make sure that you have already installed python 3.x and have internet connection. 
Keep running the script until the black cmd screen automatically gets removed. 
This script will automatically download java and cpusim. 
After complete execution, search for file name 'cpu-sim-download' something like that and run the cpu sim where written windows batch file.
If anyone face any issue , please ask
